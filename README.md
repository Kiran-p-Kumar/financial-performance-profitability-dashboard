# 💰 Financial Performance & Profitability Dashboard

## 📌 Project Overview

Organizations manage thousands of financial transactions across multiple accounts and budgets, making it challenging to monitor profitability and control expenses. This project analyzes financial transactions, budget allocations, and account-level data to deliver actionable insights into revenue, expenses, profitability, budget variance, and financial trends through an interactive Power BI dashboard.

---

## 🎯 Business Objectives

- Monitor Revenue, Expenses, and Net Profit
- Compare Budget vs Actual Spending
- Identify Major Expense Categories
- Track Monthly Financial Trends
- Support Data-Driven Financial Decisions

---

## 🛠️ Tech Stack

- Microsoft Excel (Data Cleaning)
- Python (Pandas)
- SQL
- Power BI
- DAX

---

## 📊 Dashboard KPIs

- Total Revenue
- Total Expense
- Net Profit
- Profit Margin
- Budget vs Actual
- Expense Analysis
- Monthly Revenue Trend
- Monthly Expense Trend
- Monthly Profit Trend

---

## 📈 Business Insights

- Identified the highest expense categories contributing to overall business costs.
- Compared planned budgets with actual spending to highlight budget variances.
- Monitored monthly revenue, expenses, and profitability trends.
- Enabled account-level expense analysis to identify major cost drivers.
- Delivered an executive dashboard for faster, data-driven financial reporting.
